# Ta bort mig

En minimalistisk och modern webbsida som hjälper användare att ta bort sina personuppgifter från olika webbplatser.

**Webbplats:** [delit.github.io/tabort](https://delit.github.io/tabort/) · **Repository:** [github.com/delit/tabort](https://github.com/delit/tabort)

## Funktioner

- **Minimalistisk design** - Rent och avskalat gränssnitt med mycket vitt utrymme
- **Modern styling** - Använder Poppins-font och mörkblå accentfärg (#1e40af)
- **Responsiv design** - Fungerar perfekt på alla enheter från mobil till desktop
- **Dropdown-meny** - Enkel och intuitiv navigation till olika webbplatser
- **Direktlänkar** - Tar användare direkt till rätt sida för att ta bort personuppgifter
- **SEO-optimerad** - Innehåller alla nödvändiga meta-taggar och semantisk HTML
- **Tillgänglighet** - Stöder keyboard-navigation och ARIA-attribut

## Teknisk implementation

### HTML
- Semantisk HTML5-struktur
- SEO-optimerade meta-taggar
- Open Graph-taggar för sociala medier
- ARIA-attribut för tillgänglighet

### CSS
- CSS Grid och Flexbox för layout
- CSS Custom Properties för konsistent styling
- Responsiv design med media queries
- Smooth animations och transitions
- Modern box-shadow och border-radius

### JavaScript
- Vanilla JavaScript (inga externa ramverk)
- Event listeners för dropdown-funktionalitet
- Keyboard navigation support
- Click tracking för analytics
- Error handling för externa länkar

## Keyboard shortcuts

- **Alt + D** - Öppna dropdown-meny
- **Escape** - Stäng dropdown-meny
- **Piltangenter** - Navigera i dropdown-meny
- **Enter/Space** - Välj länk i dropdown

## Responsiv design

Webbsidan är optimerad för:
- **Desktop** (1200px+)
- **Tablet** (768px - 1199px)
- **Mobil** (320px - 767px)
